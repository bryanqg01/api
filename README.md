API desarrollada con Flask que implementa operaciones CRUD sobre recursos.
Proyecto creado como práctica de aprendizaje para la materia de Sistemas de Información.

Características
CRUD completo (crear, leer, actualizar, eliminar).

-Integración con API externa de chistes de Chuck Norris.

-Traducción automática de inglés a español.

-Manejo de errores con códigos HTTP comunes (200, 201, 400, 404).

-Almacenamiento en memoria, ideal para pruebas y desarrollo.
